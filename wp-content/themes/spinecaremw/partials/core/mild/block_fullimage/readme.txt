Overview

Label: Full Image
Name: block_fullimage
Version: 1.0.0
Author: Bret Hummel


======================
Description
======================

The Full Image block can be used to display a panel with a single Image (Image Only) or Video (Video Only), or to display a panel with a background image and floating content box over the top (Image + Text). For the "Image Only" style, the image can be optionally set to full width. For videos and images that are not full-width, settings include choosing the background color.

For the "Image + Text" style, settings include whether or not up to two buttons are included.
