======================
Description
======================

This block allows for a gallery of images in a variety of configurations. 

style = columns, (bricklaid, odd couple - maybe these are specific to Sussner?)
if columns: Set number of columns (2, 3, 4)

if columns: Set whether stragglers center in container or left align
on each individual image, choose whether it takes up 1 spot or 2 (vert or horiz as well?)
