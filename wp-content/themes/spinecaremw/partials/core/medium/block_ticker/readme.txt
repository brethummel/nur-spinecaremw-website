======================
Overview
======================

Label: Photo Strip
Name: block_photostrip
Version: 1.0.0
Author: Bret Hummel


======================
Description
======================

This block will create a strip of photos that can either be static or a carousel. When used as a carousel, the "featured" photo can be highlighted and brought to the foreground.
